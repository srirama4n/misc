User

Utterance (one row per message turn, user or bot)

Channel

Conversation (a logical thread that may span sessions)
conversationId*, createdAt, resolvedAt, status, csat, nps

Context (key/value bag captured at a time)
contextId*, capturedAt, kv (map/json), source (client|server|integration)

Attachment

Event (generic analytic event)

Relationship types (direction →) & semantics

USER →STARTED→ SESSION

SESSION →HAS_UTTERANCE→ UTTERANCE (ordered by turnIndex)

UTTERANCE →REPLY_TO→ UTTERANCE (threading/quote chains)

UTTERANCE →ASSERTS_INTENT→ INTENT (confidence)

UTTERANCE →MENTIONS_ENTITY→ EntityType (value, charStart, charEnd, confidence)

UTTERANCE →ABOUT_TOPIC→ Topic (confidence)

UTTERANCE →SENT_BY→ User (when direction=USER)

UTTERANCE →GENERATED_BY→ Bot (when direction=BOT, include latencyMs, modelTokenUsage)

SESSION →ON_CHANNEL→ Channel

CONVERSATION →CONTAINS→ SESSION

USER →PART_OF→ CONVERSATION (or CONVERSATION →WITH→ USER)

CONTEXT →APPLIES_TO→ SESSION (or → UTTERANCE) (scope: session|utterance)

USER →HAS_HEALTH→ HealthMetric

UTTERANCE →HAS_ATTACHMENT→ Attachment

EVENT →RELATES_TO→ {User|Session|Utterance|Conversation} (polymorphic via separate rels or a hub)

POLICY →GOVERNS→ {User|Utterance|Attachment|Context}

Scores: csat, nps, sentiment, toxicityScore, engagementScore, frustrationScore (float)

Example ingestion (from relational rows)
// Upsert User
MERGE (u:User {userId:$user_id})
  ON CREATE SET u.createdAt = datetime($user_created_at),
                u.locale = $locale, u.timeZone = $tz;

// Upsert Session
MERGE (s:Session {sessionId:$session_id})
  ON CREATE SET s.startedAt = datetime($start_ts), s.channel = $channel
  ON MATCH  SET s.endedAt   = coalesce(s.endedAt, datetime($end_ts));

// Link User↔Session
MERGE (u)-[:STARTED]->(s);

// Utterance
MERGE (m:Utterance {utteranceId:$utt_id})
  ON CREATE SET m.createdAt = datetime($msg_ts), m.direction=$direction,
                m.text=$text, m.lang=$lang, m.sentiment=$sentiment, m.turnIndex=$turn;

// Link session to utterance
MERGE (s)-[:HAS_UTTERANCE]->(m);

// If BOT message, connect to Bot
FOREACH (_ IN CASE WHEN $direction='BOT' THEN [1] ELSE [] END |
  MERGE (b:Bot {botId:$bot_id})
  MERGE (m)-[:GENERATED_BY {latencyMs:$latency}]->(b)
);

// Intent & entities (if present)
FOREACH (i IN coalesce($intents,[]) |
  MERGE (it:Intent {intentName:i.name})
  MERGE (m)-[:ASSERTS_INTENT {confidence:i.confidence}]->(it)
);
FOREACH (e IN coalesce($entities,[]) |
  MERGE (et:EntityType {name:e.type})
  MERGE (m)-[:MENTIONS_ENTITY {value:e.value, confidence:e.confidence}]->(et)
);

MATCH (:Session {channel:$channel})-[:HAS_UTTERANCE]->(m:Utterance)-[r:ASSERTS_INTENT]->(i:Intent)

ORDER BY hits DESC;
